## 2. 대표 문제

```text
2차 지연시스템의 응답 특성을 감쇠비에 따라 구분하고, 그 특성을 비교하여 설명하시오.
```

유사 문제:

```text
감쇠비 ζ에 따른 2차 시스템의 과도응답 특성을 설명하시오.
2차 시스템에서 부족감쇠, 임계감쇠, 과대감쇠의 차이를 설명하시오.
2차 시스템의 극점 위치와 감쇠비의 관계를 설명하시오.
2차 시스템의 감쇠비가 오버슈트, 진동성, 정착시간에 미치는 영향을 설명하시오.
```

## 3. 출제 의도

이 문제는 단순히 `ζ`의 범위를 외우는 문제가 아니다. 다음 연결을 평가한다.

```text
표준 2차 시스템 모델
→ 특성방정식
→ 극점 위치
→ 감쇠비 구간
→ 과도응답 특성
→ 현장 튜닝 trade-off
```

고득점 답안은 감쇠비 구간을 표로 분류하는 데서 끝나지 않고, 각 구간의 극점 형태와 응답 특성을 현장 제어 품질과 연결해야 한다.

## 4. 문제 범위와 제외 범위

### 포함 범위

```text
- 표준 2차 전달함수
- 특성방정식
- 감쇠비 ζ
- 고유진동수 ωn
- 감쇠진동수 ωd
- 극점 위치
- ζ=0, 0<ζ<1, ζ=1, ζ>1, ζ<0 구분
- 오버슈트
- 진동성
- 상승시간
- 피크시간
- 정착시간
- 현장 튜닝 trade-off
```

### 주의해야 할 제외 범위

문제에서 말하는 “2차 지연시스템”은 일반적으로 second-order lag system, 즉 2차계 과도응답 특성을 의미한다.

다음 내용은 보조적으로만 다룬다.

```text
- dead time
- pure time delay
- e^{-Ls}
- phase margin 감소
- Smith predictor
```

시간지연은 현장 적용 또는 보완 설명으로 언급할 수 있지만, 본문 핵심이 `e^{-Ls}`로 넘어가면 문제 요구에서 벗어날 수 있다.

## 5. 정답 기준표

| 감쇠비 조건 | 명칭 | 극점 형태 | 응답 특성 | 안정성 |
|---|---|---|---|---|
| `ζ < 0` | 음의 감쇠 | 우반평면 극점 또는 양의 실수부 | 진폭 증가, 발산 가능 | 불안정 |
| `ζ = 0` | 무감쇠 | 순허수근 `s = ±jωn` | 지속진동 | 임계 안정 또는 marginal stability |
| `0 < ζ < 1` | 부족감쇠 | 좌반평면 복소켤레근 | 감쇠진동, 오버슈트 가능 | 점근 안정 |
| `ζ = 1` | 임계감쇠 | 중근, 반복 실근 | 오버슈트 없는 가장 빠른 무진동 수렴 | 점근 안정 |
| `ζ > 1` | 과감쇠 또는 과대감쇠 | 서로 다른 두 실근 | 오버슈트 없으나 느린 수렴 | 점근 안정 |

이 표는 `fact_anchor.json`에서는 `canonical_tables`로 저장되어야 한다.

## 6. 핵심 수식

### 표준 2차 전달함수

```text
G(s) = Kωn² / (s² + 2ζωn s + ωn²)
```

### 특성방정식

```text
s² + 2ζωn s + ωn² = 0
```

### 부족감쇠 극점

`0 < ζ < 1`일 때:

```text
s = -ζωn ± jωd
```

### 감쇠진동수

```text
ωd = ωn√(1 - ζ²)
```

## 7. s-평면 다이어그램이 표현해야 할 의미

ASCII 다이어그램은 모양보다 의미가 중요하다.

필수 구성요소:

```text
- 실수축 σ
- 허수축 jω
- 원점
- 좌반평면 극점
- 복소켤레 극점
- ωn
- ωd
- 실수부 -ζωn
- 각도 θ
```

올바른 의미:

```text
안정한 2차 시스템의 극점은 좌반평면에 위치한다.
부족감쇠 극점은 복소켤레 형태이다.
극점의 실수부는 -ζωn이다.
극점의 허수부는 ±ωd이다.
ωn은 원점에서 극점까지의 거리이다.
ωd는 허수축 방향 성분이다.
θ를 음의 실수축 기준으로 잡으면 ζ = cosθ이다.
θ를 허수축 기준으로 잡으면 ζ = sinθ 표현이 가능하다.
```

ASCII 다이어그램은 줄 간격과 문자 폭 때문에 위치가 왜곡될 수 있다. 따라서 그림 위치만 보고 fatal을 판단하지 않고, 그림의 축과 label, 그림 주변 설명, 수식, 표의 대응 관계를 함께 본다.

## 8. 감쇠비별 설명 기준

### ζ < 0

```text
명칭:
  음의 감쇠 또는 negative damping

극점:
  실수부가 양수인 우반평면 극점 가능

응답:
  진폭 증가, 발산 가능

판정:
  불안정
```

주의: `ζ<0`을 안정 조건으로 설명하면 치명 오류이다.

### ζ = 0

```text
명칭:
  무감쇠

극점:
  ±jωn

응답:
  감쇠 없이 지속진동

판정:
  임계 안정 또는 marginal stability
```

주의: `ζ=0`을 점근 안정 또는 하나의 값으로 수렴한다고 쓰면 오류이다.

### 0 < ζ < 1

```text
명칭:
  부족감쇠, underdamped

극점:
  좌반평면 복소켤레근

응답:
  감쇠진동
  오버슈트 가능
  ζ가 작을수록 오버슈트와 진동성이 커짐

판정:
  점근 안정
```

주의: `0<ζ<1`을 불안정 또는 발산 조건으로 설명하면 치명 오류이다.

### ζ = 1

```text
명칭:
  임계감쇠, critically damped

극점:
  중근, 반복 실근

응답:
  오버슈트 없는 가장 빠른 무진동 수렴

판정:
  점근 안정
```

주의: `ζ=1`을 과감쇠 또는 overdamped로 분류하면 치명 오류이다.

### ζ > 1

```text
명칭:
  과감쇠, 과대감쇠, overdamped

극점:
  서로 다른 두 실근

응답:
  오버슈트는 없으나 임계감쇠보다 느린 수렴

판정:
  점근 안정
```

주의: 과감쇠를 중근이라고 설명하면 치명 오류이다. 중근은 `ζ=1` 임계감쇠 조건이다.

## 10. 현장 적용과 설계 판단

고득점 답안은 감쇠비 구간을 설명한 뒤 현장 trade-off로 연결해야 한다.

```text
오버슈트:
  밸브, 모터, 위치제어, 압력제어에서 충돌·헌팅·품질 불량 원인이 될 수 있다.

상승시간:
  빠른 응답이 필요할수록 부족감쇠 영역을 허용할 수 있다.

정착시간:
  너무 작은 ζ는 진동이 길어지고 정착시간이 증가할 수 있다.

안정성:
  ζ>0 조건과 좌반평면 극점 조건을 우선 만족해야 한다.

구동기 부담:
  빠른 응답은 조작량 증가, 포화, 마모, 에너지 증가를 유발할 수 있다.

설비 보호:
  충돌 위험 또는 과압 위험이 있는 설비에서는 오버슈트를 엄격히 제한해야 한다.
```

튜닝 관점:

```text
PID 또는 서보 튜닝에서는 응답속도와 안정성 사이의 trade-off를 본다.

오버슈트가 허용되는 빠른 응답:
  부족감쇠 영역 일부 허용

오버슈트가 위험한 설비:
  임계감쇠 또는 약간 과감쇠 성향

응답속도보다 안전성이 중요한 설비:
  과감쇠 성향 허용

외란이 큰 공정:
  정착시간, 안정 여유, 조작량 한계를 함께 고려
```

비용과 적용성:

```text
정확한 감쇠비 추정에는 step test, trend 분석, system identification이 필요하다.
운전 중 시험은 생산 품질과 안전에 영향을 줄 수 있다.
기존 설비에서는 센서 추가, 데이터 수집, tuning window 확보 비용을 고려해야 한다.
고속 제어를 위해 제어주기나 통신망을 개선할 경우 기존 PLC, drive, network 호환성과 정지시간을 고려해야 한다.
```

## 13. 치명 오류 기준

### 감쇠비 구간 오류

```text
ζ=1을 과감쇠 또는 overdamped로 분류함
ζ>1을 임계감쇠로 분류함
0<ζ<1을 불안정 또는 발산 조건으로 설명함
ζ=0.707을 임계감쇠 정의로 설명함
ζ<0.7만 부족감쇠라고 제한함
0.7≤ζ<1을 과감쇠라고 설명함
```

### 극점 오류

```text
안정한 표준 2차계 극점 실수부를 +ζωn으로 설명함
s = ζωn ± jωd 형태로 쓰고 안정 극점으로 해석함
좌반평면 극점을 불안정으로 설명함
우반평면 극점을 안정으로 설명함
과감쇠를 중근으로 설명함
임계감쇠를 서로 다른 두 실근으로 설명함
```

### 응답 특성 오류

```text
임계감쇠가 진동 또는 오버슈트 응답이라고 설명함
과감쇠가 오버슈트 또는 최속 응답이라고 설명함
부족감쇠를 무조건 발산이라고 설명함
ζ=0을 점근 안정 또는 하나의 값으로 수렴한다고 설명함
```

### 각도 관계 오류

```text
θ를 음의 실수축 기준으로 정의하고 sinθ=ζ라고 설명함
θ를 음의 실수축 기준으로 정의하고 sinθ=σ/ωn이라고 설명함
```

단, θ를 허수축 기준으로 정의한 경우에는 `ζ=sinθ` 표현이 가능하다.

## 14. ASCII 표 평가를 위한 claim 추출 기준

답안자가 ASCII 표를 쓰면 표 자체를 그대로 평가하지 않는다. 먼저 다음 claim을 추출한다.

```text
condition_to_name:
  ζ=1 → 임계감쇠
  ζ>1 → 과감쇠
  0<ζ<1 → 부족감쇠

name_to_pole_type:
  임계감쇠 → 중근
  과감쇠 → 서로 다른 두 실근
  부족감쇠 → 복소켤레근

condition_to_response:
  0<ζ<1 → 오버슈트 가능
  ζ=1 → 무오버슈트 최속 수렴
  ζ>1 → 무오버슈트 느린 수렴
```

오답 claim 예:

```text
ζ=1 → over damped
over damped → 중근
ζ>1 → undamped
0<ζ<1 → unstable
```

이런 claim은 정답 기준표와 직접 충돌하므로 fatal 또는 major로 처리한다.

## 15. ASCII 다이어그램 평가를 위한 claim 추출 기준

답안자가 ASCII s-평면 다이어그램을 쓰면 다음 claim을 추출한다.

```text
axis_claim:
  σ축은 실수축이다.
  jω축은 허수축이다.

pole_position_claim:
  안정 극점은 좌반평면에 있다.
  우반평면 극점은 불안정하다.

component_claim:
  극점 실수부는 -ζωn이다.
  극점 허수부는 ±ωd이다.
  ωn은 원점에서 극점까지 거리이다.
  ωd는 허수축 방향 성분이다.

angle_claim:
  음의 실수축 기준이면 ζ=cosθ이다.
  허수축 기준이면 ζ=sinθ 표현이 가능하다.
```

오답 claim 예:

```text
안정 극점 실수부 = +ζωn
좌반평면 극점 = 불안정
우반평면 극점 = 안정
음의 실수축 기준에서 sinθ=ζ
```

다이어그램은 ASCII 배치가 깨질 수 있으므로, 그림 위치만으로 fatal 처리하지 않는다. 반드시 주변 설명과 함께 판단한다.

## 17. 사람 검토 체크리스트

```text

4. 중근이 임계감쇠 조건으로 들어갔는가?

7. ζ=0.707이 임계감쇠 정의가 아니라 실무적 타협점으로 들어갔는가?

9. ASCII 다이어그램의 부호, 축, 좌반/우반평면 오류를 잡는 logic rule이 있는가?

```
