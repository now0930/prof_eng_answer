# second_order_system_resonance_frequency_response

## 목적

2차 시스템의 주파수응답, 공진주파수, 감쇠비 조건을 평가하기 위한 topic pack이다.

## 핵심 구분

- 시간응답 overshoot와 주파수응답 resonance는 같은 개념이 아니다.
- ζ=1은 임계감쇠 조건이지 공진 조건이 아니다.
- 표준 2차 저역통과계의 공진주파수는 ωr = ωn√(1−2ζ²)이며 ζ<1/√2에서 존재한다.
